\doxysubsection{Detailed Description}
The abstract class \mbox{\hyperlink{class_hornet_engine_1_1_sound_1_1_sound_source}{Sound\+Source}} will be used to pass around a shared source. 

This ensures that the source wont have to be re-\/created and deleted with every sound sample.
